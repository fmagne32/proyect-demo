# Proyecto FastApi


## Initialize Pipenv
```
npm install

```
## PipEnv Install
```
pipenv shell

```

## start proyect  and hot-reloads for development
```
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload or  uvicorn app.main:app --host 127.0.0.1 --port 8000

```
